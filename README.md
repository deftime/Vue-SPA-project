Простий гібридний веб-додаток, побудований на Vue + статика в адмінці.

Реалізований, як повноцінний кейс юридичної онлайн-консультації. Клієнти залишають свої питання на фронтовій частині сайту, юристи логінятся у адмінку, відповідають на питання та надсилають їх клієнтам.

```
Фронтовий UI - Vue.js
Адмінка - статика, JavaScript, Bootstrap, fetch data, Firebase API.
Бек - хмарні бази даних Google Firebase, Realtime Database.
Зв'язок з поштою та розсилками - Mail API
```

Посилання репозиторію - демонстраційне.
